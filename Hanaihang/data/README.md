# Data Folder

This folder holds source configs and generated files for the data pipeline.

- `osm/areas.json` - Provinces/areas used for OSM queries
- `osm/brands.json` - Brand filters for OSM queries
- `raw/` - Raw downloads (ignored by git)
- `derived/` - Normalized JSON outputs (ignored by git)
- `directories/` - Store directory JSONs per mall (ignored by git)
